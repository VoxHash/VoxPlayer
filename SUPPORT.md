# Support

## Getting Help

- **Open an issue**: Use the appropriate template on [GitHub Issues](https://github.com/voxhash/voxplayer/issues)
- **Check documentation**: See [README.md](README.md) and [docs/](docs/) for guides
- **FAQ**: Check [docs/faq.md](docs/faq.md) for common questions
- **Contact**: Email contact@voxhash.dev

## Issue Templates

- **Bug Report**: Use the bug report template for issues
- **Feature Request**: Use the feature request template for new ideas
- **Documentation**: Use the documentation template for doc improvements

## Before Reporting

1. Check existing issues to see if your problem has been reported
2. Search the documentation for solutions
3. Try the latest version of VoxPlayer
4. Include relevant system information (OS, Python version, VoxPlayer version)
